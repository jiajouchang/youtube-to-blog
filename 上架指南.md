- [ ] **128x128 圖示** - ✅ 已有 `icons/icon128.png`

- [ ] **1280x800 截圖** - 至少 1 張（最多 5 張）

##### 商店資訊
- [ ] **簡短說明**（最多 132 字元）

- [ ] **分類**：生產力工具
- [ ] **語言**：繁體中文 + 英文
- [ ] **隱私權政策**（因為使用 storage 權限）

#### 3. 隱私權政策（必要）
因為擴充功能使用了 `storage` 權限，必須提供隱私權政策。

**簡單範本**：
```
隱私權政策

本擴充功能「YouTube 轉部落格文章」重視您的隱私：

1. 資料收集
- 本擴充功能僅在您的瀏覽器本地儲存 Gemini API Key
- 不會收集、傳輸或儲存任何個人資料到外部伺服器

2. API Key 使用
- API Key 僅用於呼叫 Google Gemini API
- 儲存在瀏覽器本地 (chrome.storage.local)
- 不會分享給第三方

3. 第三方服務
- 使用 Google Gemini API 進行文章生成
- 使用 YouTube Transcript API 取得影片字幕
- 請參閱各服務的隱私政策

4. 聯繫方式
如有隱私疑問，請聯繫：[您的email]

最後更新：2025/11/29
```

### 📊 審核時間表

- **提交審核** → 0 天
- **審核中** → 1-7 天（通常 2-3 天）
- **發布** → 審核通過後立即上架

審核重點：
- 是否符合 Chrome Web Store 政策
- 權限使用是否合理
- 是否有隱私權政策
- 功能是否如說明所述
