## 🌊 方案 3：上架到 Microsoft Edge Add-ons (免費)

Microsoft Edge 使用 Chromium 核心，因此您的擴充功能完全相容，不需修改程式碼即可上架！

### ✅ 優點
- **完全免費**：不需要支付開發者註冊費。
- **無縫遷移**：使用相同的 `.zip` 檔案。
- **潛在流量**：Edge 瀏覽器市佔率高，且競爭較 Chrome 商店少。

### 🚀 Edge 上架步驟

#### 步驟 1：註冊開發者帳號
1. 前往 [Microsoft Partner Center (Edge)](https://partner.microsoft.com/dashboard/microsoftedge/public/home)
2. 使用 Microsoft 帳號 (Outlook/Hotmail) 登入。
3. 註冊成為開發者 (Individual)，過程 **免費**。

#### 步驟 2：建立新擴充功能
1. 進入開發者儀表板 (Dashboard)。
2. 點擊 **"Create new extension"**。
3. 上傳您現有的 `youtube-to-blog-extension.zip` 檔案。
   - 系統會自動驗證 `manifest.json`。

#### 步驟 3：填寫資訊 (Store Listing)
大部分資訊可與 Chrome 商店共用：
- **Availability**: 設為 Public (公開)。
- **Markets**: 預設 All markets (全球)。
- **Store Listing Languages**: 新增 "Chinese (Traditional)" 並設為預設。
- **Description**: 複製前述的商店說明文案。
- **Logo**: 上傳 `icons/icon128.png`。
- **Promotional tiles**:
  - Small: 440x280 px (與 Chrome 共用)
  - Large: 1400x560 px (可選，建議製作以增加曝光)
- **Screenshots**: 上傳 1280x800 px 截圖 (與 Chrome 共用)。

#### 步驟 4：提交審核
1. 檢查所有資訊無誤。
2. 點擊 **"Publish"**。
3. 審核時間通常約 **1-7 個工作天**。
